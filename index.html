What It Does

This is a browser-based fugue and counterpoint MIDI generator. It creates a structured piece from sections of bars, keys, modes, metres, cadences, voice-leading rules, breath/rest choices, and weighted randomness.

The generator aims for calm, coherent, harmonically pleasing material rather than completely wild note soup. It writes a MIDI file, a project JSON recipe, and generation notes with checker warnings.

Quick Start

  1. Choose sections in Form: bars, key, mode, time signature, and cadence.
  2. Choose Fugue or Imitation + Invention, voice count, breathing, density, and pitch mode.
  3. Press Generate + Save MIDI. The MIDI file should download automatically.
  4. Use Save Project JSON to keep the exact recipe and seed for later.

Randomness

The app runs locally in the browser. It uses the browser Web Crypto random source to make a seed, then uses that seed for repeatable musical choices inside the generator. On a normal web page, JavaScript cannot read /dev/urandom directly, but modern browsers expose secure random bytes through crypto.getRandomValues. Advanced users can optionally mix in a custom entropy endpoint; see PUBLISHING.md.

Pitch Modes

Equal Temperament writes ordinary MIDI notes. This is the safest mode for most synths and DAWs. The export is note-only: separate voice tracks with note on/off events and no program changes, controller setup, tempo map, or track-name clutter.

Amy Dub Intonation writes carrier MIDI notes for a retuner such as Entonal. The MIDI note numbers may look unusual because the retuner turns those carrier keys into Amy Dub ratio pitches. This export is also note-only, so the retuner or DAW receives clean voice tracks. For the tuning logic, look at the ratios in the GitHub README.md section called Gravity Counterpoint Rules And Amy Dub Ratios.

Bend MIDI writes multiple single-voice MIDI tracks, one voice per track/channel, and retunes each voice toward Amy Dub intonation with pitch-bend messages. It should be played through separate mono instruments or separate mono channels. If the parts are merged into one polyphonic instrument, the pitch bends will fight each other and the tuning will not work properly.

Modes And Gravity

Major, harmonic minor, modal scales, and gravity melodic minor are available. Gravity melodic minor allows extra minor-key passing tones with tendency rules: leading tones rise, raised sixths lift, flat sevenths fall, and flat sixths resolve down.

Conceptual Inspirations

  • Fux-style species counterpoint and Palestrina-style voice-leading practice.
  • Knud Jeppesen-style Palestrina counterpoint as a stylistic reference.
  • Backtracking counterpoint generators that place notes, validate rules, and retry.
  • Constraint-programming approaches such as FuxCP, used here only as a design reference for explicit hard rules.
  • Markov/probabilistic Palestrina approaches such as Farbood and Schoner, used here only as inspiration for combining deterministic rules with weighted musical choices.
  • Interactive rule-checking tools such as CounterPointer, used here only as inspiration for producing a human-readable report.

Implementation Notes

  • This version was vibe coded with Codex, collaborating as Lambda Echo.
  • The MIDI writer in src/app.js is original code.
  • The moving torus visualisation uses Three.js, an MIT-licensed open-source 3D library.
  • The seeded random functions are small public-domain-style PRNG/hash patterns commonly circulated in JavaScript communities; they are included inline only for deterministic replay after the browser supplies a system entropy seed.
  • Browser APIs used: Canvas, WebGL through Three.js, optional Device Motion/Orientation input, Web Audio, optional Vibration API feedback, Blob download, TextEncoder, and Web Crypto crypto.getRandomValues.
