What Export MIDI delivers

Export MIDI saves Standard MIDI note events, plus tempo and time-signature data when Tempo Data is on. Equal Temperament MIDI follows the receiving synth tuning, usually A4 = 440; Fishtail now defaults to A3 220 Hz so ordinary synths begin in familiar tuning. If you change the reference, the receiving synth must be tuned to match. Amy Dub Intonation needs Entonal or a similar retuner to interpret its carrier notes; Bend MIDI needs separate mono channels or instruments that receive its pitch-bend data correctly.

Generate creates the piece. Export MIDI saves the music file using the project name, BPM, style, and seed. Save Project keeps the loadable Fishtail settings and form. Audio and CV exports render from the generated piece and may show a size estimate.

Open User Guide Technical Overview

What It Does

This is a browser-based imitation, invention, counterpoint, and Fishtail Fugue MIDI generator. It creates a structured piece from sections of bars, keys, modes, metres, cadences, voice-leading rules, breath/rest choices, and weighted randomness.

The generator aims for calm, coherent, harmonically pleasing material. It writes a MIDI file, a project JSON recipe, and generation notes with musical observations, form details, and kind little Gemma reassurances.

Quick Start

  1. Choose sections in Form: bars, key, mode, meter, cadence, role, and treatment.
  2. Choose Imitation + Invention, Counterpoint, or Fishtail Fugue, then set voice count, pedal voices, breathing, density, and pitch mode.
  3. Press Generate to make the piece, then open Exports and choose Export MIDI.
  4. Use Save Project to keep the form, pitch, tempo, and structure settings for later.

Pulse And Time

The Pulse + Time panel is the live sound and conductor-timing area. The page loads silent. Turning on Pulse sound starts the Teardrop reference tone at the current Reference Hz. Pulse pitch chooses the reference note, and Fine pitch trims cents by ear; both write into the same Reference Hz used by MIDI generation.

Living Reference listens only after you press Listen for pitch. Play or sing one sustained monophonic note, then press Use stable pitch to freeze the recent pitch centre as Fishtail's exact reference. Capture anyway is there when the source is intentionally moving. The input is analysed locally through a silent graph; it is not monitored, recorded, uploaded, or saved with device identity.

Metronome starts a louder pink-noise ticker using the preview meter, meter accents, and the rational swing / irrational swing controls. When Preview meter says Follow section 1, it takes the time signature from the first Form row. Rational swing moves from stroll to swagger; irrational swing adds stumble, with Lattice Safe, Hybrid Drift, and Living Drift feel modes. Swing starts straight by default; DUB mode sets a gentle strolling swing. Tempo lattice writes the same pulse timing into the MIDI conductor track as Set Tempo events, while the notes themselves stay on their formal grid.

Plain Equal Temperament MIDI exports ordinary note numbers, so exact captured acoustic tuning depends on the receiving synth. Project JSON, Pulse WAV, and Fishtail's tempo/reference metadata record the captured relationship; Amy Dub Intonation and Bend MIDI are specialised MIDI outputs that need a retuner or synth/channel setup to interpret them correctly.

Export Pulse WAV and Export Ticker WAV render optional mono 48 kHz / 24-bit stems for the generated song. Pulse WAV is a short first-two-bars reference tone; ticker WAV stays whole-piece and peak-normalized to -6 dBFS for DAW-friendly alignment. Prepare analogue CV ZIP renders the selected voice set as clock, 1V/oct pitch, gate, and calibration WAV files. Fishtail shows estimated file size and temporary render memory before audio export. The CV controls set voice package, first-60s / whole-short-piece duration, musical pulse / bar / 24 PPQN clock, full-scale volts, zero offset, gate polarity, and retrigger gap. Browser CV export defaults to one voice and the first 60 seconds for iPad memory safety. Pitch CV needs a DC-coupled interface or DAW CV utility; ordinary phone and headphone outputs usually will not preserve DC pitch voltage.

Randomness

The app runs locally in the browser. It uses the browser Web Crypto random source to make a seed, then uses that seed for repeatable musical choices inside the generator. On an ordinary web page, JavaScript cannot read /dev/urandom directly, but modern browsers expose secure random bytes through crypto.getRandomValues. Advanced users can optionally mix in a custom entropy endpoint; see PUBLISHING.md.

D4 rolls a gentler form with a tasteful chance of Refrain or Gentle Development after the first section. D20 rolls a wider form with more varied keys, meters, Refrain returns, Development rows, and occasional Dubby treatments. The first section stays Fishtail/Source so later rows have source material to return to or develop.

Pitch Modes

Equal Temperament writes ordinary MIDI notes. This is the safest mode for most synths and DAWs, but it does not retune the instrument. Fishtail's default reference is A3 = 220 Hz, which implies A4 = 440 Hz, matching the usual synth default. If you change the reference, the receiving synth must be tuned to match if you want Equal MIDI to line up with the pulse. The voice tracks stay clean: note on/off events only, no program changes, no controller setup, and no track-name clutter. If Tempo Data is on, a separate conductor track writes BPM and time signatures. The selected BPM is also shown in the downloaded filename.

Amy Dub Intonation is Amy McBride's experimental ratio-based dub tuning. Instead of dividing the octave into twelve equal steps, it maps chromatic slots to chosen ratios. It is designed to be run through Entonal or a similar retuner that can define these ratios. In retuner mode the MIDI notes are carrier keys: the note number may look ordinary or unusual, but the retuner has to receive and interpret that key as the intended ratio pitch.

Amy Dub ratio pattern, chromatic from the selected root:

  • 0 1/1 home
  • 1 16/15 upper bite
  • 2 9/8 motion
  • 3 8/7 blue low
  • 4 7/6 minor soul
  • 5 5/4 sweet light
  • 6 4/3 plagal
  • 7 11/8 shimmer
  • 8 3/2 pillar
  • 9 8/5 pivot
  • 10 13/8 tender lift
  • 11 7/4 dub crown

Bend MIDI writes multiple single-voice MIDI tracks, one voice per track/channel, with pitch-bend messages for compatible synth setups. It should be played through separate mono instruments or separate mono channels that receive pitch bend correctly. If the parts are merged into one polyphonic instrument, the pitch bends will fight each other and the tuning will not work properly.

Fishtail Gravity

Fishtail Gravity is Amy's compositional idea for making complex theory flow gently. Notes are treated as forces with tendencies: leading tones want to rise, certain minor-key colours want to fall, cadences pull the voices home, bass tones create weight, and silence lets another voice answer.

The generator combines that gravity with counterpoint, fugue structure, rhythmic breathing, and optional Amy Dub ratios. Equal Temperament keeps the usual twelve-step tuning; Amy Dub Intonation keeps the same musical functions but lets the pitch world bend toward Amy's ratio experiment.

Modes And Gravity

Major, harmonic minor, modal scales, and gravity melodic minor are available. Gravity melodic minor allows extra minor-key passing tones with tendency rules: leading tones rise, raised sixths lift, flat sevenths fall, and flat sixths resolve down.

Fishtail Fugue uses the Form rows as tonal regions, then shapes them into exposition, episodes, middle entries, final return, and cadence. If the form is too short, the generator quietly adds enough internal sections or bars and explains the shaping in Generation Notes. This means a fugue can come out longer than the visible Form. With DUB off it uses Formal Gravity; with DUB on it keeps the fugue map but lets the bass and offbeat answers breathe more freely.

Dub Gravity keeps the counterpoint engine but biases the bass toward repeated root/fifth gravity, gives middle voices offbeat skank gestures, makes the upper voices breathe more, and allows the occasional parallel-perfect bend in the rules while noting it gently in Generation Notes. The interface also shifts into a black and green terminal-style view while DUB is armed.

Refrains, Pedals, And Held Notes

Role chooses what each Form row does. The first row is always Fishtail, because it introduces the source material. Later Refrain rows return that first-section idea, and later Development rows reshape it into a new answer. If a Development ever has no source to work from, it writes a fresh Fishtail section and says so in Generation Notes.

Treatment chooses the flavour of a Refrain or Development. Source marks the first Fishtail row. Clear Return keeps a Refrain close to the first-section source. Gentle develops it with key movement, octave changes, small answering gestures, light rhythm changes, and voice handoff. Dubby keeps recognizable fragments while the bass finds more root/fifth gravity and the inner voices answer with offbeat skank chords.

Pedal voice switches tell the generator that a voice is allowed to hold longer tonic or fifth notes as an intentional drone. Bass pedal turns on by default in DUB mode. For melody and inner voices, long held notes are shaped to breathe and resolve musically, so they feel like expressive tension rather than accidental stuck notes.

Project JSON remembers section roles, treatments, pedal choices, Refrain source material, and Development transformations, so a favourite form can be saved and returned to later.

Conceptual Inspirations

  • Fux-style species counterpoint and Palestrina-style voice-leading practice.
  • Knud Jeppesen-style Palestrina counterpoint as a stylistic reference.
  • Backtracking counterpoint generators that place notes, validate rules, and retry.
  • Constraint-programming approaches such as FuxCP, used here only as a design reference for explicit hard rules.
  • Markov/probabilistic Palestrina approaches such as Farbood and Schoner, used here only as inspiration for combining deterministic rules with weighted musical choices.
  • Interactive rule-checking tools such as CounterPointer, used here only as inspiration for producing a human-readable report.
